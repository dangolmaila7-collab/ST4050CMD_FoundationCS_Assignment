# Task 1 – Secure Data Exchange

## Overview
This task explores secure data exchange mechanisms, focusing on:
- SMTP with TLS
- Encoding methods (Base64 vs ASCII)
- Common attack vectors
- Authentication with OAuth and JWT

## Contents
- Diagrams
  - SMTP/TLS flow
  - Encoding comparison
  - Attack vectors
  - OAuth/JWT flow
- Scripts
  - Base64 encoder/decoder (Python demo)

## How to Run Scripts
1. Navigate to this folder:
   cd Task1_SecureDataExchange
2. Run the Python script:
   python base64_demo.py

## Notes
- Diagrams illustrate secure communication processes and vulnerabilities.
- Scripts demonstrate practical encoding/decoding.
- This task highlights the importance of encryption and authentication in secure data exchange.
